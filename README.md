# CodeAlpha Task 3 - Data Visualization

## Project Overview

This project focuses on visualizing S&P 500 company data using Python and Matplotlib. The objective is to transform raw data into meaningful visual representations that help identify trends and patterns.

## Tools Used

* Python
* Pandas
* Matplotlib

## Dataset

The dataset contains information about S&P 500 companies, including:

* Company Symbol
* Company Name
* Sector
* Headquarters Location

## Visualizations Created

### 1. Number of Companies by Sector

A bar chart showing the number of companies present in each sector.

### 2. Sector Distribution

A pie chart illustrating the percentage share of each sector within the S&P 500.

### 3. Top 10 Headquarters Locations

A bar chart displaying the locations with the highest concentration of company headquarters.

## Key Insights

* Industrials, Financials, and Information Technology are the most represented sectors.
* New York City has the highest number of company headquarters.
* The distribution of companies across sectors is not uniform.
* Major business hubs account for a significant portion of company headquarters.
